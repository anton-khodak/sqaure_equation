# Розв'язувач квадратних рівнянь

Дає повний розв'язок квадратного рівняння виду 
```
AX^2 + BX + C = 0
```
де A, B, C - параметри, які задаються користувачем

## Встановлення та запуск

Вимагає Python 3

#### З командного рядка
```
python main.py
```

#### Браузер
```bash
pip install -r requirements.txt
python server.py
```
